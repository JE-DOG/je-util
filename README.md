# je-util

`je-util` is a CLI for automating routine work of JE-DOG

## Commands

- `notion_daily_report` - generate a Notion daily report
- `pkttn` - import Kaiten tasks into a Notion database

## Install

### From source

```bash
git clone https://github.com/JE-DOG/je-util.git

./tools/scripts/sync.sh
```

## Run

```bash
je-util --help
```
